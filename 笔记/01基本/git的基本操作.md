1.clone（远程克隆到本地）

从远程将库克隆到本地(强迫症可以先cd到克隆的位置)（可能会出现仓库为空的警告）

git clone '地址'


2.add（暂存）

在新建操作完后添加到库（操作后要cd要库的目录内）

git add 新建的文件名称


3.commit（暂存确认后提交到本地仓库）

若单独提交某文件可以加上<文件名>，否则全部提交 ，操作说明必须要加

git commit <文件名> -m '操作说明'


4.push（提交到远程）

将本地仓库提交到远程仓库

git push



5.pull（覆盖本地）

可以将本地已有的远程库覆盖到本地

git pull


6.checkout（检出）

切换分支或恢复工作树文件

git checkout 恢复文件名
